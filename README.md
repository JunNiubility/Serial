# Serial

#### 介绍
串口助手和气象上位机

#### 软件架构
软件架构说明

#### 使用说明

1.  包含串口助手，但是这个串口助手目前只能支持8位数据的串口收发
2.  CDIO上位机界面
3.  串口助手与上位机界面的串口相互独立，互补影响
